# Website Institucional ÁGIL ENERGIA

Este projeto é o site institucional da ÁGIL ENERGIA, desenvolvido utilizando HTML5, CSS3 (Vanilla) e JavaScript Vanilla (ES6+). O objetivo é criar um site moderno, elegante e responsivo, focado na identidade visual e nas funcionalidades específicas da empresa no setor de gás natural onshore no Brasil.

## Funcionalidades Principais:

- **Design Responsivo:** O site é totalmente adaptado para diferentes tamanhos de tela (mobile-first).
- **Bilíngue (PT/EN):** Conteúdo alternável entre português e inglês via JavaScript.
- **Animações e Interatividade:** Utilização de CSS e JavaScript para transições suaves, animações on-scroll e micro-interações.
- **Performance:** Lazy loading de imagens e otimização de animações.
- **Acessibilidade:** Implementação de HTML semântico e práticas ARIA.

## Estrutura do Projeto:

- `index.html`: Arquivo principal HTML com a estrutura do site.
- `style.css`: Arquivo CSS com a estilização completa do site.
- `script.js`: Arquivo JavaScript para a lógica de interatividade e funcionalidade bilíngue.
- `assets/`: Diretório para armazenar imagens, vídeos e outros recursos (a ser criado conforme necessário).

## Como Rodar o Projeto:

1. Clone o repositório.
2. Abra o arquivo `index.html` em seu navegador.

## Paleta de Cores:

- Principal: Laranja vibrante (`#F5A623`)
- Secundária: Roxo profundo (`#4A1D6E`)
- Neutros: Branco, cinza claro, cinza escuro
- Acentos: Gradientes suaves entre laranja e roxo
